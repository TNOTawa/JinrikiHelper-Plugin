# JinrikiHelper 插件仓库

欢迎来到 JinrikiHelper 的导出插件，在这里你可以下载或更新来自社区的插件。
你也可以投稿自己的插件，Fork本仓库并按 `插件介绍模板` 提交 PR 即可。

---

## **simple_export.py** - 简单单字导出插件
> 内置
> 最新版本：1.1.0（已在本地版1.0及后续版本内置）

- 每个字导出为一个wav文件，最简单的导出方式。

## **utau_oto_export.py** - UTAU oto.ini 导出插件
> 内置
> 最新版本：1.2.0（已在本地版1.1及后续版本内置）

- 生成 UTAU 音源配置文件，一个 wav 可包含多条配置，还包含了中日文的自动拼字功能，以及日文的CVVC音源导出。
